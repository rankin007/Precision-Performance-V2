# Precision Performance: Brand Specification

## Core Values
- Precision, Authority, Results, No Guessing.

## UI Components
- **Buttons**: Rounded corners (full), Healing Gold background.
- **Cards**: Laboratory Slate with subtle 1px Paddock Green borders.
- **Alerts**: No "Red" for errors; use "Muted Slate" for out-of-range to keep the clinical calm.

## Interaction
- All "Healing Zone" hits must trigger a Gold Glow effect.
