# Project 005: Authentication & Portal Entry

## Objective
Implement a secure, role-based authentication system using Supabase Auth to distinguish between Trainers and Clients.

## Requirements
- **Unified Login:** A single entry point that redirects users based on their `user_metadata` or profile role.
- **Brand Aesthetic:** A minimalist, "Elite" interface that aligns with the "Go by the Numbers, No Guessing" philosophy.
- **Security:** CSRF protection and secure session management via Next.js Middleware.
- **Password Recovery:** Integrated flow for lost credentials.
