# Project 002: Trainer Portal UI/UX Design

## Objective
Design a high-performance, interactive dashboard for trainers to input equine bio-data and monitor horse health via graphical displays.

## Requirements
- **Dashboard:** At-a-glance view of the stable's current health status.
- **Input Forms:** Fast, mobile-optimised entry for Urine/Saliva Brix, pH, Temperature, and Weight.
- **Visualisations:** Time-series line charts for tracking biological shifts over training cycles.
- **Gallery:** Profile-specific photo galleries for visual health tracking (e.g., coat condition, eye clarity).
- **Logistics:** Fields for 'Distance Travelled' and 'Race/Training Times' to correlate with bio-results.
