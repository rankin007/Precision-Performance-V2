# Project 014: Trainer Onboarding & Quick Start Guide

## Objective
Create a clear, concise guide that instructs trainers on how to accurately measure, convert, and log equine bio-data into the system.

## Requirements
- **Measurement Protocol:** Simple steps for collecting and testing urine/saliva.
- **The Magic Formula:** Clear explanation of the Salts conversion ($C = ms \times 1.43$).
- **Target Visualization:** A table showing the "Healing Range" targets for Sugars, Salts, and pH.
- **Qualitative Logging:** Encouraging the use of Hydration and Feeding notes for a complete health picture.
- **Brand Alignment:** Professional, elite, and calming aesthetic.
