# Technical Stack & Environment

## Primary Environment
- **Antigravity:** The core development and orchestration environment.

## External Integrations
- **GitHub:** Source control and repository management.
- **Supabase:** Backend-as-a-Service (PostgreSQL database, Auth, and Storage for horse photos).
- **Vercel:** Frontend hosting for the web portals.
- **Railway:** Hosting for any external computational or background workers.

## Core Features to Build
1. **Multi-tenant Portals:** Separate secure views for Trainers and Clients.
2. **Graphical Displays:** Time-series charts for Brix, pH, and performance metrics.
3. **Data Ingestion:** Mobile-responsive forms for temperature, weight, and comments.
