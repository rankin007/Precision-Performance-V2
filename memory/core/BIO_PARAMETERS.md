# Bio-Analysis Parameters & Formulae

## 1. Carbohydrates / Total Sugars (Brix)
- **Operational Range:** 0 – 13%
- **Target Healing Range:** 3% – 4%

## 2. Salts / Conductivity (C)
- **Input Unit:** ms (microsiemens)
- **Output Unit:** C (Conductivity number)
- **Operational Range:** 0 – 70C
- **Target Healing Range:** 15C – 20C
- **Conversion Formula:** $$C = ms \times 1.43$$

## 3. pH (Urine & Saliva)
- **Operational Range:** 0 – 10
- **Target Healing Range:** 6.4 – 7.0 pH
