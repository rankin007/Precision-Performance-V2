# Bio-Analysis Parameters (RBTI Foundation)

This file defines the data points required for every equine entry. Agents must ensure all inputs align with these metrics.

## Urine Analysis
- **Sugars (Brix):** Measuring carbohydrate metabolism and energy availability.
- **pH:** Tracking the anionic/cationic balance and digestive efficiency.
- **Conductivity (Salts):** Monitoring electrolyte balance and kidney stress.
- **Nitrates/Urea:** Analysing protein utilisation and cellular debris (Albumin).

## Saliva Analysis
- **pH:** Used in conjunction with urine pH to determine the "Biological Ionization" state of the horse.

## Physical & Performance Metadata
- **Temperature:** Daily baseline for health monitoring.
- **Weight:** Tracking mass fluctuations against training intensity.
- **Distance/Time:** Logs for training gallops or race days.
- **Event Results:** Correlation between bio-states and finishing positions.
